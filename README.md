# AWS Image Resizer Project

## 📌 Description
This project automatically resizes images using AWS S3 and Lambda.

## ⚙️ Technologies Used
- AWS S3
- AWS Lambda
- Python

## 🚀 How It Works
- Upload image to S3 bucket
- Lambda function triggers
- Image gets resized and stored

- Learned serverless architecture
